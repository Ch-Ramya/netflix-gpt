# Netflix Clone

A feature-rich Netflix-inspired web application built with **React.js**, **Redux Toolkit**, **Firebase**, and the **TMDB API**. This project demonstrates a scalable component-based architecture, secure authentication, movie previews, multi-language support, and state management using Redux Toolkit.
